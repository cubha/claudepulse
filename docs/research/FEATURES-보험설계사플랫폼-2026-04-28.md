# 보험설계사 플랫폼 (가칭 InsureAgent) — 단계별 실행 가능한 기능 & 비용 정리

작성일: 2026-04-28
원본 리서치: RESEARCH-보험설계사플랫폼-2026-04-27.md

---

## 한 줄 요약

개인 개발자가 만들 수 있는 가장 현실적인 형태는 **"GA·보험사에 묶이지 않는 설계사 종합 업무 SaaS"**.
마이데이터 직접 연동·앱 내 청약은 영구 회피, 대신 **PDF 업로드 + OCR + 보험사 코드 관리 + 분급 시뮬레이터**가 무기.
**MVP는 5~7주에 출시**, 월 인프라비 약 **8만원**부터 시작. 토스 보험파트너 종료(2025.12)로 10만 설계사가 대체 도구를 찾는 타이밍.

---

## Phase 1 — MVP / 베타 무료 출시 (5~7주, 약 2개월차 도달)

### 사용자(설계사)가 할 수 있는 것

- **가입·로그인**: 이메일, 카카오, 구글 중 하나로 가입. 비밀번호 재설정 가능. (휴대폰 본인확인은 아직 없음)
- **고객 관리(CRM)**:
  - 고객 등록·수정·삭제
  - 이름·전화번호로 검색
  - 태그 붙이기, 메모 남기기
  - 가족관계 연결(예: 김철수의 배우자=박영희)
- **계약 관리**:
  - 보험사·상품명·증권번호·만기일·납입주기·보장 요약을 직접 입력
  - 증권 PDF 또는 이미지 첨부 (자동 파싱은 아직 X — Phase 2)
- **자동 알림(이메일)**:
  - 만기 D-30 / D-7 자동 발송
  - 고객 생일·결혼기념일 자동 발송
- **디지털 명함**:
  - 본인 프로필 페이지 URL 발급
  - QR 코드 다운로드
  - 카카오톡 공유 시 미리보기 카드 자동 노출
- **기초 보장분석 계산기**:
  - 연령·소득·가족 입력 → 권장 보장액 공식 계산
  - 기존 가입 내역과 격차 시각화 (보험사별 정밀 매트릭스는 Phase 2)
- **모바일 사용**: 모바일 브라우저에서 그대로 사용 가능 (반응형 웹). 앱 설치는 아직 X.
- **약관·개인정보**: 표준 처리방침·이용약관 동의/철회 UI 제공.

### 이 단계에서 안 되는 것 (사용자에게 미리 알릴 것)

- 유료 결제, 휴대폰 본인확인
- 카카오 알림톡, SMS 발송 (이메일만 가능)
- 보험증권 PDF 자동 파싱(OCR)
- 분급제 수수료 시뮬레이터
- 모바일 앱 설치 (PWA·네이티브 모두 아직 X)
- 보험사 자동로그인, 견적·상품 비교

### 운영자 비용

| 항목 | 비용 |
|---|---|
| Vercel Pro (호스팅) | 월 약 27,600원 |
| Supabase Pro (DB·인증·스토리지·서울 리전) | 월 약 34,500원 |
| 도메인 (.com 1개) | 연 25,000~50,000원 (월 환산 약 2,000~4,000원) |
| Resend 이메일 발송 | 월 3,000통까지 무료 |
| Sentry 에러 추적 | 월 5,000건까지 무료 |
| PostHog 분석 | 월 100만 이벤트까지 무료 |
| **월 합계** | **약 80,000원** |
| 1회성 비용 | **0원** (사업자등록·변호사 자문 모두 Phase 2로 미룸) |

> 베타 30명 무료 사용자가 "써볼만하다"고 느끼면 Phase 2로.

## Phase 2 — 첫 유료 결제 받기 (추가 8~10주, 누적 약 3.5~5개월차 도달)

### 사용자가 추가로 할 수 있게 되는 것

- **월 구독 결제**: 토스페이먼츠로 카드 정기결제. 영수증·환불 처리.
- **휴대폰 본인확인**: 첫 결제 시 1회. 이후 재인증 불필요.
- **카카오 알림톡으로 알림 받기**: 만기·기념일·상담 안내가 카톡으로 도착. SMS도 옵션.
- **보험증권 PDF 자동 입력 ★**:
  - 고객이 마이데이터 앱(보맵·토스·뱅크샐러드)에서 받은 PDF를 업로드
  - OCR이 보험사·상품·증권번호·만기·보장 자동 인식 → 계약 등록 1초 완성
  - **마이데이터 본허가 없이 같은 효과를 내는 핵심 우회 기능**
- **분급제 수수료 시뮬레이터 ★★**:
  - 계약별 1년차·2~7년차 수수료 타임라인 시각화
  - 환수 리스크(고객 해지 시 토해내야 할 금액) 미리 보기
  - **시장에 같은 기능 제공하는 도구 없음 — 가장 큰 무기**
- **견적·상품 비교**: 사용자가 선택한 보험사들끼리 수동 DB 기반 비교 (라이선스 부담 없는 범위)
- **모바일 앱처럼 설치(PWA)**: iOS Safari/Android Chrome에서 "홈 화면에 추가"로 앱 아이콘 생성.
- **개인정보 강화**: 주민번호·이름·전화는 별도 암호화 테이블에 분리(pgcrypto), 5년 감사 로그(pgaudit) — 사용자 눈에는 안 보이지만 GA B2B 진입 신뢰도 ↑.

## Phase 3 — 차별화 무기 / GA B2B 진입 (추가 16~22주, 누적 약 7~9개월차)

### 사용자가 추가로 할 수 있게 되는 것

- **토스 보험파트너 이탈 마이그레이션 ★**: 토스에서 받은 고객 CSV를 한 번에 가져오기. (2025.12 종료로 10만 설계사가 옮겨갈 곳을 찾는 타이밍)
- **계약자≠피보험자 보험료 수동 보강**: 마이데이터에서도 안 잡히는 사각지대(자녀·부모 명의 계약)를 직접 입력 → 자동 합산.
- **분급 환수 7년 트래킹·알림**: Phase 2 시뮬레이터에 알림 기능 확장. 환수 발생 즉시 푸시.
- **보험사 코드/비밀번호 통합 관리(1단계)**:
  - 평균 20개 보험사 사원번호·비밀번호를 한 화면에서 관리
  - 1단계는 비밀번호 매니저 수준 (자동로그인 고도화는 Phase 4)
- **이직 시 고객 데이터 이관 ★**:
  - 고객 동의 위임 워크플로우로 안전하게 이전 회사 → 새 회사 데이터 이동
  - 시장 어느 앱에도 없는 기능
- **모바일 네이티브 앱(iOS/Android)**:
  - Expo 기반, App Store·Play Store 등록
  - 푸시 알림, 카메라로 즉시 증권 OCR
- **외근 오프라인 작업**: 지하·저신호 환경에서도 입력 가능, 신호 복귀 시 자동 동기화(PowerSync).
- **약관 AI 질의응답**: 약관 PDF 업로드 → "이 특약 보장되나요?" 같은 자연어 질문에 즉시 답변(RAG).
- **소개인 합법 연결 워크플로우 ★**:
  - 자격 없는 지인에게 광고비·포인트 모델로 합법 보상 (현금 모집수수료 직접 지급은 영구 금지)
  - 보험업법 §99 회피 설계 — 변호사 자문 추가 필수
- **N잡 설계사용 AI 상담 보조**: 상품·약관·고객 상황별 스크립트 자동 생성.
- **고객 셀프 데이터 공유**: 고객이 본인 마이데이터 결과를 동의 후 설계사에게 공유 → 본허가 없이 같은 효과.

## Phase 4 — 인증·시장 공인 / 마이데이터 제휴 (추가 6~12개월, 누적 약 13~21개월차)

### 사용자가 추가로 할 수 있게 되는 것

- **마이데이터 자동 동기화**: 보맵/뱅크샐러드/토스 중 1곳과 제휴 → 고객 가입정보 자동 업데이트. (자체 본허가는 끝까지 안 받음, 자본 5억 회피)
- **보험사 자동로그인(헤드리스)**: 20개 보험사 사이트 자동 로그인 — 단, 보험사별 약관 위배 여부 신중 검토 필요.
- **GA 본부 매니저용 B2B 대시보드**: 위촉·교육 이수·실적 통합 관리. 중소 GA에 SaaS 라이선스 판매.

| 매출 비례 | PG 매출의 **3.4%** |

#### 1회성 비용

|---|---|---|
| ISMS-P 간편인증 (B2B 신뢰도용) | **600~1,100만원** + 6~12개월 준비 | 매출 100억 또는 GA B2B 본격 진입 시 |
| (선택) 법인 보험대리점 설립 | 자본금 **5천만~1억** + 영업보증금 **3억** + 등록비 20만 | 직접 GA 운영 시만 — 보통 불필요 |

> ISMS-P는 매출이 100억 이상이거나 MAU 100만 이상일 때 의무. 그 전엔 비효율.

## 단계별 진입 신호 (다음 Phase로 넘어갈 타이밍)

- **Phase 1 → 2**: 베타 30명 중 절반 이상이 매주 1회 이상 로그인 + "유료여도 쓰겠다" 5명 이상 확보
- **Phase 2 → 3**: 유료 30명 돌파 + 토스 보험파트너 이탈 문의 확인 + 월 매출 100~300만
- **Phase 3 → 4**: 중소 GA 1~3곳 파일럿 진행 중 + 월 매출 1,000만+ + B2B에서 ISMS 요구 시작
