## 1. 프로젝트 개요

- **목적**: 30초 내외 브라우저 미니게임. 참가자 이름을 넣으면 캐릭터들이 경쟁하며 **1등 또는 꼴등**이 당첨되는 파티게임
- **타겟**: 친구/팀원/회식 자리에서 즉흥 사용, 유튜버/스트리머 방송용
- **핵심 요구사항**:
  - 30초 내외 게임 시간
  - 병맛(B급 유머) 요소 필수
  - 1등과 꼴등이 언제든 역전 가능한 긴박감
  - 설치 없이 URL 하나로 즉시 접근
- **레퍼런스**: [lazygyu 마블 룰렛](https://lazygyu.github.io/roulette/) — Box2D WASM 물리 기반 구슬 레이스

## 3. 아키텍처 패턴 리서치 (architecture-researcher)

### 3-1. 역전 메커니즘 패턴 분석

| 패턴 | 설명 | 30초 적합성 | 사례 |
|---|---|---|---|
| **아이템 편향 지급** | 꼴등에게 강력 아이템 집중 | 높음 | 마리오카트 블루쉘 |
| **Late-Game 카오스 주입** | 종료 5~8초 전 랜덤 이벤트 | 매우 높음 | Fall Guys |
| **물리 불확실성** | 충돌/반동으로 비결정적 결과 | 매우 높음 | lazygyu 룰렛 |
| **규칙 반전** | "지금부터 꼴등이 1등!" 선언 | 높음 | Mario Party Reverse Minigame |

**핵심 설계 원칙**: 카오스 이벤트 주입은 게임 종료 5~8초 전이 최적. 너무 이르면 회복 가능, 너무 늦으면 반응 시간 부족.

### 3-2. 30초 게임 루프 타임라인

```
[0~3초]   시작 팡파레 + 캐릭터 등장
[3~20초]  스킬/물리 기반 경쟁 (순위 형성)
[10초]    순위 현황 리마인더 (리더보드 잠깐 표시)
[20~25초] 카오스 이벤트 1회 랜덤 발동 (역전 기회)
[25~28초] BGM 1.5배 가속 + 빨간 틴트 + 순위 전체 공개
[28~30초] 화면 진동 + 카운트다운 + 슬로우모션 0.7x
[30초]    최종 판정 → 1등 폭죽 / 꼴등 낙뢰 연출
```

### 3-3. 긴장감 연출 검증 기법

- **근사 순위 UI**: 정확한 순위 대신 "상위권/중위권/하위권"만 표시 → 마지막 3초에 정확 순위 공개
- **얼리 쇼크 프라이밍**: 시작 5초 내 작은 카오스 1회 → "언제 또 터질까" 긴장 유지
- **생리적 각성**: 이벤트 타이밍 불예측 → 심박수 상승 효과

### 3-4. 병맛 요소 설계 패턴

| 카테고리 | 예시 | 병맛 강도 |
|---|---|---|
| 물리 계층 | 중력 반전, 캐릭터 10배 확대, 마찰 제거 | 상 |
| 규칙 반전 | 순위 반전 선언, 점수 음수 구간, 조작 반전 | 상 |
| 시각/청각 과장 | 1등 황금 폭죽, 꼴등 낙뢰+슬픈 트롬본, 역전 시 화면 회전 | 중상 |
| 랜덤 장애물 | 거대 물체 낙하, 반중력 버블, 충돌 반동 3배 | 중 |

### 4-2. 시장 기회 (갭 분석)

기존 제품들이 **놓치고 있는 영역**:

1. **관람형 + 역전 메커니즘 부재**: lazygyu 룰렛은 꼴등 당첨 지원하나 참가자 개입 불가. "내가 뭔가를 했는데 역전당하는" 긴박감 없음
2. **한국어 + 한국 직장문화 밈 게임 공백**: 사다리타기 외 회식/술자리 즉흥 사용 가능한 한국어 파티게임 사실상 없음
3. **설치 장벽**: Fall Guys/Pummel Party는 뛰어나지만 설치 필수 + 유료
4. **30초 이내 짧은 긴박감 게임**: 대부분의 파티게임은 라운드당 수분 소요

### 4-3. 차별화 포인트

- **"설치 없이 URL 하나로"** → 회식 자리에서 폰 꺼내 바로 가능
- **"30초 안에 드라마 완성"** → SNS 공유 가능한 짧고 강렬한 서사
- **"1등도 꼴등도 될 수 있다"** → 기존 뽑기 앱과 달리 게임 내내 긴장 유지
- **"한국 직장인 밈 특화"** → 부장, 야근, 회식, 치킨, 카카오톡 등 공감형 테마
- **"역전 이벤트의 극적 연출"** → SNS 바이럴 포인트

## 5. 팀 리더 리서치 — 게임 주제 아이디어 TOP 5

에이전트 결과를 종합하여, **30초 + 역전 + 병맛 + 1등꼴등**에 최적화된 게임 주제를 선별했다.

### TOP 1. 삼겹살 뒤집기 챔피언십 (추천도: ★★★★★)

> "적절한 타이밍에 고기를 뒤집어 최고 굽기를 달성하라! 단, 화로 온도가 갑자기 바뀐다..."

- **게임 방식**: 각 참가자에게 고기 한 점 배정. 실시간으로 고기 색이 변하며, 클릭 타이밍에 뒤집기. 최종 굽기 점수로 순위 결정
- **역전 메커니즘**: "알바생 실수" 이벤트(랜덤 고기 강제 뒤집기), "화로 폭발" 이벤트(전체 굽기 단계 상승), "사장님 특제 소스"(꼴등 고기에 보너스)
- **병맛 요소**: 고기가 날아다니는 물리, 태운 고기에서 연기 이펙트, "완벽한 굽기" 달성 시 고기가 빛나는 신성한 연출
- **왜 1위인가**: 한국 문화 보편성(삼겹살), 조작 단순(클릭 타이밍), 물리+타이밍 하이브리드로 역전 자연스러움, 구현 난이도 낮음

### TOP 2. 회사 야근 달리기 (추천도: ★★★★★)

> "퇴근 버튼을 연타해서 1등으로 퇴근하라! 단, 부장님 카톡이 오면 멈춘다..."

- **게임 방식**: 참가자 캐릭터가 횡스크롤 달리기. 화면 연타로 가속
- **역전 메커니즘**: "부장 카톡 알림"(1등 2~3초 정지), "철야 미션"(꼴등 터보 부스트), "회식 강요"(상위 3명 속도 50% 감소)
- **병맛 요소**: 직장인 밈(야근, 부장, 회식), 과로사 게이지, 퇴근 성공 시 무지개 이펙트
- **왜 2위인가**: 한국 직장인 공감도 최고, 연타 게임은 관전도 재미있음. 다만 삼겹살 대비 "연타 피로감" 존재

### TOP 3. 오징어게임 무궁화꽃 클릭 (추천도: ★★★★☆)

> "인형이 뒤돌아 있을 때만 클릭! 볼 때 클릭하면 후진!"

- **게임 방식**: 화면 상단 인형이 랜덤 타이밍에 뒤돌아봄. 뒤돌아 있을 때만 클릭해 전진, 잘못 클릭하면 벌점 후진
- **역전 메커니즘**: "터보 인형"(뒤돌아보는 속도 2배 → 선두 실수 유발), "바나나 껍질"(1등 랜덤 후진), "안전구역 이동"(꼴등에게 무적 3초)
- **병맛 요소**: 인형 표정 변화, 잡힌 순간 슬로우모션, 과장된 탈락 연출
- **왜 3위인가**: 글로벌 인지도(오징어게임), 규칙이 직관적. 다만 IP 연관성 이슈 존재할 수 있음

### TOP 4. 복불복 밥상 (고추냉이 러시안 룰렛) (추천도: ★★★★☆)

> "N개 밥그릇 중 고추냉이 그릇을 피하라! 그런데 그릇이 중간에 섞인다..."

- **게임 방식**: 각자 그릇 선택 → 셔플 애니메이션 → 뚜껑 오픈 → 고추냉이 선택자가 꼴등
- **역전 메커니즘**: "주방장 실수"(셔플 중 고추냉이 위치 추가 변경), "VIP 그릇"(황금 숟가락 = 자동 1등)
- **병맛 요소**: 뚜껑 열리는 슬로우모션, 고추냉이 발견 시 매운맛 이펙트(화면 빨개짐), 당첨자 울상 표정
- **왜 4위인가**: 심리전 + 운의 조합이 재미있음. 다만 "클릭 한 번" 게임이라 30초 동안의 긴장감 유지가 약할 수 있음

### TOP 5. 치킨 vs 피자 최후의 한 조각 (반응속도 대결) (추천도: ★★★☆☆)

> "마지막 치킨 한 조각이 등장하는 순간 클릭! 가장 빠른 사람이 1등!"

- **게임 방식**: 랜덤 대기 → 음식 등장 순간 클릭 → 반응시간 측정 → 순위 결정
- **역전 메커니즘**: "가짜 치킨" 페인트(먼저 클릭하면 벌점), "연장전"(꼴등에게 추가 기회, 결과 덮어쓰기)
- **병맛 요소**: 페인트 이미지(브로콜리, 피망), 치킨 쟁탈전 연출
- **왜 5위인가**: 반응속도 테스트로 명확하지만, 30초를 채우기 어려움 (한 판에 5초 이내). 멀티 라운드 구성 필요

```
[Phase 1: 0~20초] 스킬 경쟁 구간
  - 물리/타이밍 기반 경쟁
  - 근사 순위만 표시 (상위/중위/하위)
  - 플레이어 간 상호작용으로 자연 카오스

[Phase 2: 20~25초] 카오스 이벤트 (1회 랜덤 발동)
  - 중력 반전 / 꼴등 부스터 / 위치 셔플 / 1등-꼴등 교환

[Phase 3: 25~28초] 피날레
  - BGM 1.5배 가속 + 빨간 틴트 + 순위 전체 공개

[Phase 4: 28~30초] 카운트다운
  - 화면 진동 + 슬로우모션 0.7x

[판정] 1등: 황금 폭죽 + 왕관 / 꼴등: 낙뢰 + 슬픈 트롬본
```

| 리스크 | 대응 |
